Publier les modifications

Toutes vos modifications sont enregistrées localement dans ce navigateur. Pour les rendre visibles à tous, exportez le fichier site.json et déposez-le dans le dossier data/ du dépôt.

Mode de publication

Pour que les modifications soient visibles publiquement, deux options :

  1. Manuel (recommandé pour démarrer) : exportez site.json, remplacez-le dans data/site.json du dépôt GitHub, et committez. Le site sera mis à jour automatiquement après le déploiement.
  2. Automatisé : branchez ce backoffice à un hébergeur (Netlify CMS, Decap, ou une petite API Node/PHP) pour publier en un clic. Voir la documentation du dépôt pour la mise en place.
